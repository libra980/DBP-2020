## ❤개발 환경 소개

### why mariaDB?
웹사이트 제작을 위해 php, html 그리고 mariaDB를 이용했다.
개발자의 입장에서, mariaDB와 MySQL이 비슷한 환경이라고 하지만, 나중에 취직하거나 DB를 상업적으로 사용할 일이 생길거 같아서 MariaDB를 택했다. 
그리고 가상머신 리눅스 서버에 접속, 개발의 불편함을 해소하기 위해 vscode 를 이용했다.
